CAN FD Offset Optimizer 免安装版
================================

1. 请先完整解压 ZIP，不能直接在压缩包中运行。
2. 双击 CANFDOffsetOptimizer.exe 启动；不需要安装 Python 或任何依赖。
3. 程序会在 EXE 同级目录使用：
   - user_input：每次导入的工程副本和导入清单；
   - user_output：日志、图表、CSV、优化后 DBC 副本等结果。
4. 请把整个程序文件夹放在当前用户有写权限的位置，例如桌面或工作目录。
   不要放在 Program Files 等受保护目录。
5. 升级版本时，请先退出程序，再保留或复制旧版本的 user_input 和 user_output。
6. 原始用户 DBC、ARXML、YAML 和 XLSX 不会被程序直接修改。

发布目标：Windows 10/11 x64
